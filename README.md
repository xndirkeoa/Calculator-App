## ⚙️🚀 Installation

```bash
# 📥 Clone repo
git clone https://github.com/xndirkeoa/Calculator-App.git

# 📂 Enter folder
cd Calculator-App

# ⚙️ Compile
javac *.java

# ▶️ Run
java Main
```

---

## 🎮🧠 Usage

| 🎯 Action         | 💡 Description |
| ----------------- | -------------- |
| 🔢 Enter Numbers  | 0–9 keypad     |
| ➕ Choose Operator | +, −, ×, ÷     |
| 🟰 Calculate      | Press =        |
| 🧹 Reset          | AC             |
| 🔁 Toggle Sign    | +/-            |
| 📊 Percentage     | %              |
| 🔍 Square Root    | √              |
